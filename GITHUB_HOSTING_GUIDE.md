# 🚀 Step-by-Step Guide: Host Your App for Free on GitHub Pages

Since your computer doesn't have Git installed in the command line yet, the easiest, most beginner-friendly way to host your site on GitHub is using **GitHub Desktop**. It handles installation, login, and commits visually without any command-line typing.

---

## Step 1: Install GitHub Desktop (Easiest way for Windows)
1. Go to [desktop.github.com](https://desktop.github.com/) and download the installer.
2. Run the installer and sign in with your GitHub account (create one first at [github.com](https://github.com) if you haven't already).

---

## Step 2: Add your project folder to GitHub Desktop
1. Open **GitHub Desktop**.
2. Click **File** (top left) ➔ **Add Local Repository...** (or press `Ctrl + O`).
3. Click **Choose...** and select your project folder:
   `C:\Users\SAKTHIVEL\Desktop\HF`
4. Since it is not a Git repository yet, GitHub Desktop will show a message: 
   *"This directory does not appear to be a Git repository. Would you like to create a repository here instead?"*
5. Click **create a repository** (in blue text).
6. Fill in the popup details:
   - **Name**: `healthy-homes-foods` (or any name you like)
   - Keep everything else as default.
   - Click **Create Repository**.

---

## Step 3: Publish to GitHub
1. You will see a list of your files on the left sidebar.
2. In the bottom-left box, type a summary title like `Initial Commit` and click the blue **Commit to master** (or **Commit to main**) button.
3. Click the **Publish Repository** button at the top.
4. **Important**: Uncheck the box that says *"Keep this code private"* (GitHub Pages requires the repository to be **Public** to host it for free).
5. Click **Publish Repository**. Your files are now uploaded to GitHub!

---

## Step 4: Enable GitHub Pages (Continuous Hosting)
1. Open your browser and go to your repository page on GitHub (e.g. `https://github.com/your-username/healthy-homes-foods`).
2. Click the **Settings** tab (gear icon at the top of the repository page).
3. On the left sidebar menu under "Code and automation", click on **Pages**.
4. Under **Build and deployment** ➔ **Branch**:
   - Change `None` to `master` (or `main`).
   - Leave `/ (root)` as is.
   - Click **Save**.
5. Wait about 30 seconds. Refresh the page.
6. A box will appear at the top showing your live URL:
   ➔ **"Your site is live at https://your-username.github.io/healthy-homes-foods/"**

---

## 📱 Running on Android
Open this live URL on your phone's browser. It is secure (`https`), hosted continuously, updates in real-time, and is accessible from any device anywhere in the world!
